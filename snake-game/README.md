## 🐍 Snake Game (Java Swing)

This is a classic Snake Game built using **Java Swing**, focusing on basic game development concepts such as event handling, rendering graphics, movement logic, collision detection, and score tracking.  
It is a small but complete desktop game that demonstrates clean logic building and 2D game programming fundamentals.

## 🎮 Features

- Smooth snake movement using arrow keys  
- Random apple generation  
- Score tracking  
- Growing snake body after eating apples  
- Collision detection (self + walls)  
- Game Over screen  
- Simple and lightweight Java Swing UI  

## 🛠️ Technologies Used

- **Java**
- **Java Swing**
- **AWT Graphics**
- **Event Handling (KeyAdapter, Timer)**

## 📂 Project Structure

snakegame/
 ├── Board.java       # Main game board logic (movement, rendering, collision)
 ├── SnakeGame.java   # Game window setup (JFrame)
 └── icons/           # Images for apple, snake head, and dot


## ▶️ How to Run

1. Install **JDK 8+**
2. Compile: javac snakegame/*.java
3. Run: java snakegame.SnakeGame
